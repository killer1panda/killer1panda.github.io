## 🔧 TROUBLESHOOTING

**Page won't load?**
- Make sure you're using the full filepath: `file:///Users/ajay/Downloads/RaviKlaassens_Offline/index.html`
- Browser must allow local file access (most do by default)

**Images not showing?**
- This is expected in offline mode for external CDN images
- Your portfolio images on works/ page should all show
- If not, check browser console (F12) for specific errors

**Navigation not working?**
- Make sure you're using `file://` protocol
- Try refreshing the page with Cmd+R
- Check that you haven't moved the RaviKlaassens_Offline folder
